## 📖 In Breve

Un'introduzione pratica a Python per principianti assoluti. La prima metà copre i fondamenti della programmazione (variabili, loop, funzioni, classi). La seconda metà mette in pratica questi concetti attraverso tre progetti concreti: un gioco arcade, visualizzazioni di dati e un'applicazione web.
