# 🔰 Python Essentials 1

## Obiettivo del Corso

Questo corso è un percorso di apprendimento strutturato verso la **certificazione PCEP-30-0x (Certified Entry-Level Python Programmer)** dell'OpenEDG Python Institute, in partnership con Cisco Networking Academy.
